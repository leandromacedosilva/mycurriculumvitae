# 📄 Meu Currículo Profissional | Web Portfolio

Este repositório contém o código-fonte do meu currículo profissional, desenvolvido de forma responsiva utilizando **HTML5** e **CSS3**. O projeto foi reestruturado com foco em design corporativo moderno (Corporate Tech), aplicando conceitos de clean code, semântica estrutural e tipografia otimizada para leitura.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estrutura semântica (`<header>`, `<main>`, `<ul>`, etc.) otimizada para acessibilidade e leitores de tela.

* **CSS3:** Uso de variáveis nativas (`:root`) para gerenciamento de paleta de cores executiva, flexbox para alinhamentos precisos e pseudo-elementos (`::before`) para estilização personalizada.

* **Google Fonts:** Integração da família tipográfica *Inter*, padrão atual de mercado para interfaces de alta legibilidade.

Atuar como **Analista de TI** ou **Desenvolvedor Front-End**, aplicando sólida experiência em infraestrutura corporativa, gerenciamento de sistemas operacionais e administração de bancos de dados relacionais, combinada com o constante aprimoramento em tecnologias de desenvolvimento web.

### 💡 Dicas de RH para o seu GitHub:
1. **Ative o GitHub Pages:** Vá em *Settings (Configurações)* do seu repositório no GitHub, procure a seção *Pages*, escolha a branch `main` (ou `master`) e clique em salvar. Isso vai gerar um link público para o seu currículo.

2. **Adicione o Link no Topo:** Pegue esse link gerado e coloque-o na descrição principal do repositório (na engrenagem ao lado de "About" na página inicial do projeto). Assim, o recrutador pode clicar e ver o seu currículo renderizado na hora!
